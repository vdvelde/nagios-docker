# nagios-docker
An Enterprise ready docker for Nagios including Nagiosgraph, nrpe, nrdpe

Current version are included :
- nagios-4.1.1
- nagios-plugins-2.1.1
- nrpe-2.15
- nagiosgraph-1.5.2
- nrdp
- radar

The aim is to setup Nagios for monitoring including the above plugins.
